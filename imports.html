Introdução

Bem-vindo ao tutorial import! import é usado no python para adicionarmos bibliotecas ao nosso código, bibliotecas são algoritimos que já foram feitos e podem ser usados em nosso código.

Com elas, podemos:

  • ✅ criar codigos mais complexos.
  • ✅ adicionar funções não existentes no python.
  • ✅ até mesmo conectar nosso codigos importando um para o outro.

como exemplo vamos usar as bibliotecas:

  • math – usado para operações complexas de matematica.
  • random – usado para criar uma pseudo aleatoriedade.
  • time – Permite usarmos o tempo real no codigo.

math

math vamos usar essa biblioteca para simplificar um calculo e assim agilizar nosso desenvolvimento neste caso vamos fazer a raiz quadrada:

import math # aqui estamos importando a biblioteca math

x = math.sqrt(64) # depois de importar a biblioteca, podemos usar suas funções,
                    nesse caso a raiz quadrada (que é sqrt)
                
print(x) # e por fim, vamos printar o resultado

random

random é utilizado para criar algo proximo a aleatoriedade, como sabemos o computador não é capaz de fazer algo verdadeiramente aleatorio mas atraves de uma serie de contas matematicas podemos criar algo que parece aleatorio.

import random # novamente importamos a biblioteca desejada
print(random.randint(1, 10)) # ao usarmos a função random estamos chamando a biblioteca 
                               com isso podemos pedir um numero aleatorio inteiro (randint) entre 1 e 10

time

time essa biblioteca serve para podermos utilizar o tempo no nosso codigo, não so podemos marcar horas mas também podemos fazer outras coisas como definir intervalos de tempo:

import time

print("ola.") # aqui estamos apenas imprimindo uma mensagem imediatamente ao rodar o codigo
time.sleep(2.4) # aqui estamos definindo um intervalo de tempo de 2.4 segundos
print("mundo.") # e aqui estamos imprimindo outra mensagem apos o intervalo de tempo
