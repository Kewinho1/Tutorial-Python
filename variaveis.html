Tutorial: Como Trabalhar com Variáveis em Python 🐍💡

Objetivo🎯🔍

Neste tutorial, você aprenderá a criar e usar variáveis em Python. Vamos entender como armazenar dados em variáveis, trabalhar com tipos de dados básicos (como inteiros, strings e floats) e realizar operações simples. Este é um tutorial ideal para quem está começando com Python e quer entender os fundamentos de armazenamento e manipulação de dados.

O que são Variáveis em Python?

Em Python, variáveis são usadas para armazenar dados temporários. Essas variáveis podem ser utilizadas para armazenar diversos tipos de informações, como números, texto ou até resultados de cálculos.

Exemplo simples de variável:

nome = "João"
idade = 25

nome é uma variável que armazena o valor "João".
idade é uma variável que armazena o valor 25.

1. Criando Variáveis

Para criar uma variável em Python, basta escolher um nome para ela e atribuir um valor usando o operador =.

nome = "Maria"
idade = 30
altura = 1.65

nome recebe o valor "Maria", uma string.
idade recebe o valor 30, um inteiro (int).
altura recebe o valor 1.65, um número de ponto flutuante (float).

2. Tipos de Dados Básicos em Python

  • Inteiros (int) 💯: Números inteiros, positivos ou negativos, sem decimais. Exemplos: 5, -10, 200.
  • Strings (str) 💬: Textos, ou sequências de caracteres, entre aspas. Exemplos: "olá", "Python", "123".
  • Floats (float) 🌊: Números decimais. Exemplos: 3.14, -0.5, 100.0.

3. Usando Variáveis em Operações 🔧

Operações com Inteiros:

numero1 = 10
numero2 = 5

# Soma
soma = numero1 + numero2
print(soma)  # Resultado: 15

# Subtração
diferenca = numero1 - numero2
print(diferenca)  # Resultado: 5

Operações com Floats:

peso = 70.5
altura = 1.75

# Cálculo do IMC
imc = peso / (altura ** 2)
print(imc)  # Resultado: 23.02 (aproximadamente)

Concatenando Strings:

nome = "Maria"
sobrenome = "Silva"

# Concatenando strings
nome_completo = nome + " " + sobrenome
print(nome_completo)  # Resultado: "Maria Silva"

4. Alterando o Valor das Variáveis 

idade = 25  # Definindo a idade inicialmente
print(idade)  # Resultado: 25


# Alterando o valor da variável idade
idade = 26
print(idade)  # Resultado: 26

5. Exibindo os Valores das Variáveis 

nome = "Pedro"
idade = 20

# Exibindo os valores das variáveis
print("Nome:", nome)
print("Idade:", idade)

📊Resultado:

Nome: Pedro
Idade: 20
