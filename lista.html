Tutorial: Listas em Python: Como Criar e Manipular Simples 📝🔢

Objetivo 🎯

Neste tutorial, você aprenderá a criar e manipular listas em Python. Vamos entender como armazenar múltiplos itens em uma lista, acessar esses itens e realizar operações simples, como adicionar, remover e modificar valores. Este é um tutorial essencial para quem deseja entender como trabalhar com coleções de dados em Python.

O que São Listas em Python? 🧳

Em Python, uma lista é uma coleção ordenada de itens. Você pode armazenar vários tipos de dados em uma lista, como números, strings, ou até outras listas.

Exemplo de Lista:

numeros = [1, 2, 3, 4, 5]

1. Criando uma Lista ✍️

frutas = ["maçã", "banana", "laranja"]

2. Acessando Itens de uma Lista 🔍

frutas = ["maçã", "banana", "laranja"]

# Acessando o primeiro item
print(frutas[0])  # Resultado: maçã

# Acessando o segundo item
print(frutas[1])  # Resultado: banana

Dica: Índices negativos

# Acessando o último item da lista
print(frutas[-1])  # Resultado: laranja

3. Modificando Itens de uma Lista ✏️

frutas = ["maçã", "banana", "laranja"]

# Modificando o segundo item
frutas[1] = "kiwi"
print(frutas)  # Resultado: ['maçã', 'kiwi', 'laranja']

4. Adicionando Itens a uma Lista ➕

Usando append:

frutas = ["maçã", "banana", "laranja"]

# Adicionando ao final
frutas.append("manga")
print(frutas)

Usando insert:

frutas = ["maçã", "banana", "laranja"]

# Inserindo na segunda posição
frutas.insert(1, "morango")
print(frutas)

5. Removendo Itens de uma Lista ❌

Usando remove:

frutas = ["maçã", "banana", "laranja", "kiwi"]
frutas.remove("banana")
print(frutas)

Usando pop:

frutas = ["maçã", "banana", "laranja", "kiwi"]
removido = frutas.pop(2)
print(frutas)
print("Removido:", removido)

Usando del:

frutas = ["maçã", "banana", "laranja", "kiwi"]
del frutas[0]
print(frutas)

6. Comprimento da Lista 📏

frutas = ["maçã", "banana", "laranja", "kiwi"]
comprimento = len(frutas)
print(comprimento)

7. Ordenando uma Lista 🔀

Usando sort:

frutas = ["maçã", "banana", "laranja", "kiwi"]
frutas.sort()
print(frutas)

Usando sorted:

frutas = ["maçã", "banana", "laranja", "kiwi"]
lista_ordenada = sorted(frutas)
print(lista_ordenada)
