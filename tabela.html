📊 Tutorial de Tabelas com Pandas

Aprenda a criar e editar tabelas com facilidade no Python

📦 1. Instalando o pandas

Abra o terminal e execute o seguinte comando:

pip install pandas

🧱 2. Criando uma Tabela Simples (DataFrame)

import pandas as pd

dados = {
    'Nome': ['João', 'Maria', 'Pedro', 'Ana'],
    'Idade': [28, 22, 35, 24],
    'Cidade': ['São Paulo', 'Rio de Janeiro', 'Belo Horizonte', 'Curitiba']
}

tabela = pd.DataFrame(dados)

print(tabela)

Resultado:

    Nome  Idade            Cidade
0   João     28        São Paulo
1  Maria     22   Rio de Janeiro
2  Pedro     35  Belo Horizonte
3    Ana     24          Curitiba

➕ 3. Adicionando e Modificando Colunas

tabela['Salário'] = [3000, 3500, 4000, 2500]
tabela['Idade'] = tabela['Idade'] + 1
print(tabela)

Resultado:

    Nome  Idade            Cidade  Salário
0   João     29        São Paulo     3000
1  Maria     23   Rio de Janeiro     3500
2  Pedro     36  Belo Horizonte     4000
3    Ana     25          Curitiba     2500

🔍 4. Filtrando Dados

filtro = tabela[tabela['Idade'] > 30]
print(filtro)

Resultado:

    Nome  Idade            Cidade  Salário
2  Pedro     36  Belo Horizonte     4000

🧹 5. Removendo Colunas

tabela = tabela.drop('Salário', axis=1)
print(tabela)

Resultado:

    Nome  Idade            Cidade
0   João     29        São Paulo
1  Maria     23   Rio de Janeiro
2  Pedro     36  Belo Horizonte
3    Ana     25          Curitiba

📈 6. Ordenando os Dados

tabela = tabela.sort_values(by='Idade')
print(tabela)

Resultado:

    Nome  Idade            Cidade
1  Maria     23   Rio de Janeiro
3    Ana     25          Curitiba
0   João     29        São Paulo
2  Pedro     36  Belo Horizonte
