Tutorial: Repetindo Tarefas com For e While em Python 🔁🐍

Objetivo 🎯

Neste tutorial, você aprenderá a usar os laços de repetição em Python: for e while. Vamos entender como criar loops para iterar sobre listas, usar range() e até fazer repetições até que uma condição seja atendida. Esse é um conceito fundamental em programação para automatizar tarefas repetitivas!

O que são Laços de Repetição? 🔄

Permitem executar uma parte do código várias vezes sem repeti-lo manualmente. Os principais laços em Python são for e while.

1. Laço for - Para Iterar sobre Sequências 🔢

frutas = ["maçã", "banana", "laranja"]

for fruta in frutas:
    print(fruta)

Saída:

maçã
banana
laranja

2. Laço for com range() - Repetições Numéricas 🔢

for i in range(1, 6):
    print(i)

Saída:

1
2
3
4
5

3. Laço while - Repetição até uma Condição ⏳

contador = 1

while contador <= 5:
    print(contador)
    contador += 1

Saída:

1
2
3
4
5

4. Laço for com Condições e if ⚙️

frutas = ["maçã", "banana", "laranja", "kiwi"]

for fruta in frutas:
    if fruta == "banana":
        print("Encontramos uma banana!")
    else:
        print("Outra fruta:", fruta)

Saída:

Outra fruta: maçã
Encontramos uma banana!
Outra fruta: laranja
Outra fruta: kiwi

5. Controle de Fluxo: break e continue ⚡

Exemplo com break:

for i in range(1, 10):
    if i == 6:
        print("Número 6 encontrado! Parando o loop.")
        break
    print(i)

Saída:

1
2
3
4
5
Número 6 encontrado! Parando o loop.

6. Laços Aninhados 🐣

for i in range(1, 4):
    for j in range(1, 3):
        print("i = {}, j = {}".format(i, j))

Saída:

i = 1, j = 1
i = 1, j = 2
i = 2, j = 1
i = 2, j = 2
i = 3, j = 1
i = 3, j = 2
