Funções em Python

Introdução

Funções são blocos de código reutilizáveis que realizam uma tarefa específica. Elas ajudam a organizar o código, evitar repetições e facilitar a manutenção.

Com funções, podemos:

  • ✅ Reutilizar código facilmente.
  • ✅ Dividir o programa em partes menores e mais gerenciáveis.
  • ✅ Facilitar a leitura e manutenção do código.

Definindo Funções

Para definir uma função em Python, usamos a palavra-chave def, seguida do nome da função e parênteses.

def saudacao():
    print("Olá! Bem-vindo ao tutorial de funções.")

# Chamando a função
saudacao()

Parâmetros e Retorno

Funções podem receber parâmetros para trabalhar com dados de entrada e podem retornar valores usando a palavra-chave return.

def soma(a, b):
return a + b
            
resultado = soma(5, 3)
print"A soma é:",resultado

Chamando Funções

Para executar uma função, basta chamá-la pelo nome seguido de parênteses, passando os argumentos necessários.

def saudacao():
    print("Olá! Bem-vindo ao tutorial de funções.")

def soma(a, b):
    return a + b

# Chamando as funções
saudacao()
print(soma(10, 20))

Exemplo Completo

Veja um exemplo completo que utiliza funções para calcular a área de um retângulo:

def area_retangulo(largura, altura):
return largura * altura
            
largura = float(input("Digite a largura: "))
altura = float(input("Digite a altura: "))
            
print"A área do retângulo é ",area_retangulo(largura, altura)
